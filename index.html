FAQ's

What do you mean by a consumer grade CPU ?

A Quad-core cpu with AVX2 instructions. Currently we support only x86_64 architectures.

What about languages SDK other than python ?

Unfortunately for now, it is difficult to support each language SDK separately. But we are working on releasing C ABI based SDK allowing users to integrate them in their language of choice, given almost all languages have some C interoperability.

What can I do with it ?

Since SDK implements fast Computer-vision based algorithms, you should be able to use for cases where Computer-vision can help. For more use-cases please checkout tutorials.

What personal data do you store ?

Only personal information we store is your email and hashed-password whenever you signup. All payment-related information is processed by Level 1 PCI DSS Compliant Razorpay. Your credit card or PayPal credentials are never directly handled by or stored on Ramanlabs.in servers.

What are Licensing Conditions ?

End user License allows purchased SDK to be used only for personal purposes. You can however run it on any personal number of devices you own , because basically it is just a binary file that runs fully local.
